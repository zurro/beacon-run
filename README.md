# Beacon Runner

A **3-lane infinite runner** with score chasing, power-ups, tutorial onboarding, a local leaderboard, and a faster `Super` difficulty.

Play it here: https://zurro.github.io/beacon-run/index.html

## Controls
- **Left / Right**: change lanes
- **Up**: jump floor breaks and bars
- **Space**: pause
- **Enter**: start or restart
- **Mobile**: swipe left / right / up

## Power-ups
- **SHIELD**: survive one hit
- **MAGNET**: pulls pickups toward you
- **BOOST**: temporary speed increase

## Rewards
- **COIN** trails: `+100` score per coin
- **BEACON** item: `+500` score
